Detailed Description

Abstract singleton initialisator class, needed for everything :P.

Just one instance per process must exitst. To instantiate it use libathome_client::Init::Init() or ::libathome_server::Init::Init() constructor. Use libathome_common::Common::get() static method to get the singleton instance from anywhere in your program. Also implicitly includes STL stuff, stdlibs, etc ...
