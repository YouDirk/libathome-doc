Detailed Description

Base class to throw excpetions.

Inherit from this class to implement your exception. So we can catch (libathome_common::Error& e) {} all exceptions in one statement.

To throw this class, use the macro throw Err("error message!"); defined in libathome-common/Error.hpp. This makes sure that the Class::method() name is included in the Error::what() message.

This class does also provides a backtrace functionality. The backtrace will be automatically added to Error::what(), if debug build is enabled by defining the preprocessor constant DEBUG, i.e. passing the flag -DDEBUG through the C++ compiler.

Example

try {
...
if (error_case)
throw Err("This is an %s message!", "error");
...
} catch (Error& e) {
Log->error(e);
}
