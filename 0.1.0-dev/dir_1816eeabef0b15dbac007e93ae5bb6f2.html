Detailed Description

Holds all files for namespace libathome_client.

Include libathome-client.hpp (implicitly includes libathome-common.hpp) to use all header files in this directory and thereby all the classes using the namespace libathome_client which represents the API of libathome-client.so or libathome-client.dll.

// Incudes all files from directory 'libathome-client/' and implicitly
// 'libathome-common/'
#include <libathome-client.hpp>
