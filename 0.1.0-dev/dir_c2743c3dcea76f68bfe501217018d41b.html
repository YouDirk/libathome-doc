Detailed Description

Holds all files for namespace libathome_common.

Include libathome-common.hpp to use all header files in this directory and thereby all the classes using the namespace libathome_common which represents the API of libathome-common.so or libathome-common.dll.

// Incudes all files from directory 'libathome-common/'
#include <libathome-common.hpp>
